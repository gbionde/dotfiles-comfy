## Build
**Distro:** Arch Linux <br/>
**WM:** qtile <br/>
**Compositor:** jonaburg-picom fork <br/>
**Terminal:** kitty <br/>
**Shell:** zsh + oh-my-zsh <br/>
**Dependencies:** feh, scrot, amixer, brightnessctl, setxkbmap, thunar, ttf-nerd-fonts-symbols. <br/>
